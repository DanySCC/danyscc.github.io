**What it does:**
Analyzes execution feasibility for business opportunities through 3 lenses:

**1. Skills Matching**
- Maintains inventory of available skills:
  - Technical: Node.js, Python, AI/ML, APIs, databases, automation, web dev
  - Business: Market research, sales, marketing, product management
  - Domain: E-commerce, automation, AI agents, SaaS, trading systems
- Compares required skills vs available skills
- Calculates match score (0-1)
- Identifies critical skill gaps

**2. Competitive Advantage Assessment**
- Analyzes 4 moat types:
  - **Technical moats:** Proprietary tech, AI models, algorithms
  - **Market moats:** Network effects, brand, customer lock-in
  - **Operational moats:** Cost structure, speed, quality
  - **Resource moats:** Data, partnerships, capital
- Compares vs competitors from knowledge base
- Calculates advantage score (0-1)

**3. Execution Feasibility**
- Checks resource availability:
  - Time (hours available per week)
  - Money (budget for tools/services)
  - Skills (team capabilities)
  - Infrastructure (servers, APIs, tools we already have)
- Estimates build timeline
- Identifies blockers and dependencies
- Returns feasibility score (0-1)

**Key functions:**
- `analyzeSkills(description, opportunity)` - Skills gap analysis
- `assessCompetitiveAdvantage(description, competitors)` - Moat analysis
- `calculateFeasibility(description, resources)` - Execution analysis
- `synthesizeAnalysis(skillsAnalysis, advantageAnalysis, feasibilityAnalysis)` - Combines all findings
