## üìä SCORING METHODOLOGY

Each problem scored on 5 dimensions (1-10 scale):

1. **Revenue Impact** - Direct money impact on the business
2. **Urgency** - How quickly this bleeds money/loses customers
3. **Market Size** - How many SMEs have this problem
4. **Automation Difficulty** - (Lower = easier to automate, 1=trivial, 10=very hard)
5. **Competitive Advantage** - Defensibility and differentiation potential

**ROI Score Formula:**
```
ROI = (Revenue √ó Urgency √ó Market √ó (11-Difficulty) √ó Competitive) / 10000
```

This formula weights:
- **High revenue impact** (direct money)
- **High urgency** (pain NOW)
- **Large market** (many potential customers)
- **Low difficulty** (easier/faster to build = better ROI)
- **High competitive advantage** (sustainable moat)
